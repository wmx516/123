## 核心知识点
- HTML 文档的基本骨架：<!DOCTYPE html> → <html> → <head> → <body>
- 标题标签：<h1> ~ <h6>，<h1> 为最大标题
- 文本格式：<p>、<strong>、<em>、<del>、<mark>、<br>、<hr>
- 列表：无序列表 <ul> + <li>，有序列表 <ol> + <li>
- 图片：<img src="..." alt="...">，src 用于指定图片路径，lt 用于替代文本
- 链接：<a href="..." target="_blank">，href 指定跳转地址，	arget="_blank" 在新标签页打开
- 表格：<table>、<tr>、<th>（表头）、<td>（单元格）
- 表单：<form>、<input>（text / password / email / radio / checkbox / submit）、<textarea>、<select>、<button>、<label>
- 语义化标签：<header>、<nav>、<main>、<section>、<article>、<footer>
- HTML5 新特性：<audio>、<video>、<details> / <summary>、placeholder、
equired

## 错题与 Insights
- src vs href：src 用于嵌入资源（图片、脚本），href 用于引用链接地址
- lt 不仅是图片失败时的 fallback，对视障用户和 SEO 也非常重要
- <main> 一个页面通常只用一次，表示主要内容区域
