一、变量的定义

JavaScript 中使用 var、let、const 三种方式声明变量。
var：函数作用域，可重复声明；let：块级作用域，不可重复声明；const：块级作用域，声明常量（不可重新赋值）。

二、变量类型

JavaScript 有 7 种基本数据类型：Number、String、Boolean、Undefined、Null、Symbol、BigInt， 以及引用类型 Object（包括 Array、Function 等）。
使用 typeof 运算符可以检测变量类型。

三、表达式

表达式由操作数和运算符组成，包括：算术运算符、赋值运算符、比较运算符、逻辑运算符、三元运算符等。

四、流程控制

流程控制包括：if/else 条件判断、switch/case 分支、for 循环、while 循环、break 和 continue。

五、函数

函数是可复用的代码块，支持：函数声明、函数表达式、箭头函数、默认参数、递归调用等。
